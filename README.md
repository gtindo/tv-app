# TV APP

Free app for Watching tv online

## How to use

Prerequiste :
- Nodejs runtime (v10+)
- npm or yarn installed

Installation:
- get the source code : 

```shell
git clone https://github.com/gtindo/tv-app
```

- Install dependencies :

```shell
cd tv-app
npm install
```

- Run in development mode:
```shell
npm run dev
```

- Build for production:
```shell
npm run build
```
